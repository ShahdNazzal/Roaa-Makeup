# Roaa Makeup Artist Website

This is a clean, vanilla HTML/CSS/JS project tailored for Roaa Makeup Artist. I have structurally split the original code into logical blocks while retaining absolute 1-to-1 visual representation against your requested file.

## Setup
Simply open `index.html` in your browser. Since the project uses vanilla HTML, CSS, and JavaScript, no build step or node environment is strictly required. It utilizes Tailwind CSS via CDN seamlessly.

## File Structure
- `index.html`: The main page content and components. 
- `css/style.css`: Custom styling additions safely extracted out of the HTML document.
- `js/main.js`: Main Javascript containing scroll animations, gallery filtering and form submission.

## Working with VS Code
1. Open this `makeup` directory in VS Code.
2. We recommend installing the "Live Server" extension if you want your browser to auto-reload upon making changes. Simply right-click on `index.html` and hit `Open with Live Server`.
3. Edit your HTML, CSS, or JS, and your changes will be accurately reflected.

## Exporting to GitHub
1. Create a new empty repository on GitHub.
2. Open terminal in this directory and initialize git (`git init`).
3. Add all files (`git add .`) and commit your work (`git commit -m "Initial commit"`).
4. Run the remote addition command from your GitHub Repo and `git push -u origin main` or master.
